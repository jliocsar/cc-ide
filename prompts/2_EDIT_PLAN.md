I want the Plans listed in the side bar to be editable.

Right now when I open a Plan tab, I can only suggest changes on it.

I want to be able to edit the plan itself:
- It must support both Normal/VSCode-styled and Vim-styled keybinds
    - For this, let's add an option in Ctrl+K called "Open Settings"; It should open a modal with settings to customize; The first and only one (for now) will be the "Editor keybinds", to pick between these 2 options
- So then it must support 2 different modes: Edit and Review
    - In Edit mode, mouse actions edit/select the text itself
    - In Review mode, mouse actions select lines to add comments etc
    - The "Review comments" sidebar in the right side should be collapsable
    - Let's think the best UX as to how and where we change between these modes etc

This gives me the ability to edit/tweak the plan myself, but also review the plan and request comments/changes on it in the way it currently works.
